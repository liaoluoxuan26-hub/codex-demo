# 短视频课程文案拆解器

这是一个面向手机端的前端网页小工具，用来把已经提取出来的短视频课程文字，快速整理成适合复盘和二次创作的文案结构。

## 功能

- 粘贴抖音、快手、B站等视频链接，方便记录素材来源。
- 粘贴已经提取出来的视频文字、字幕或口播稿。
- 点击“开始拆解”后，页面会以卡片形式把文字整理成以下模块：
  - 课程标题
  - 核心主题
  - 主要观点
  - 关键案例
  - 金句摘录
  - 可执行步骤
  - 适合发短视频的文案版本
- 如果只粘贴视频链接，没有粘贴文字，页面会提示：第一版暂不支持自动解析链接，请先粘贴文案内容。
- 自动过滤明显乱码、过短句子、重复句子、token、长串 ID 和无意义符号，让输出更干净。
- 课程标题会结合高频词和核心句子生成，避免直接截取开头文字。
- 主要观点优先保留完整、有观点的中文句子。
- 金句摘录优先选择有冲突、有启发、有明确观点的原文句子。
- 可执行步骤会整理成行动清单，不只是复制原文。
- 短视频文案版本包含开头钩子、正文讲解和结尾引导，更接近抖音/快手口播。
- 可以点击“一键复制全部结果”，把所有拆解模块复制到剪贴板。
- 每个结果卡片都有“复制”按钮，可以单独复制某一个模块。
- 可以点击“清空内容”，快速清空输入和结果。
- 可以点击“示例文案”，自动填入一段测试文案，方便快速体验。
- 页面使用响应式布局，适配手机屏幕，按钮和卡片之间保留舒适间距。

## 限制

- 第一版只做前端原型和基础文字拆解功能。
- 暂不支持自动解析抖音、快手、B站等平台的视频链接。
- 暂不调用 AI 接口，拆解结果基于简单的关键词和句子规则生成。
- 不需要后端、不需要数据库、不需要登录。

## 使用方法

1. 用浏览器打开 `index.html`。
2. 在“视频链接”输入框中粘贴视频链接。
3. 在“已提取的视频文字”文本框中粘贴字幕、口播稿或课程文案。
4. 如果想快速体验，也可以先点击“示例文案”。
5. 点击“开始拆解”。
6. 在“拆解结果”区域查看整理后的文案模块。
7. 如需复用结果，点击“一键复制全部结果”。
8. 如需重新填写，点击“清空内容”。

## 文件说明

- `index.html`：页面结构。
- `style.css`：手机端样式和响应式布局。
- `script.js`：输入校验、乱码和重复内容过滤、文字拆解、单模块复制和结果渲染逻辑。
